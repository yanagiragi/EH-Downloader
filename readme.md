# Gentleman-Downloader

![](https://i.imgur.com/cLk0Xwz.png)

## Current Supports

* e-hentai / exhentai (Fallbacks to e-hentai)

* Ahri8

* Wnacg

## Commands

* Download: 
    
    ```node bin/cli.js $url [$url]```

* Search: 

    ```node bin/search.js $keyword  [$keywords]```
